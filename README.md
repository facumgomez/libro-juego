# Libro-Juego

Este repositorio contiene el proyecto **Libro-Juego**, una aplicación web interactiva desarrollada con HTML, CSS y JavaScript como ejercicio práctico de frontend.

El objetivo de este proyecto fue construir una experiencia interactiva donde el usuario puede avanzar por un “libro” o juego narrativo, tomando decisiones a través de la interfaz web.

---

## 📌 Descripción

**Libro-Juego** es una aplicación web estática que permite al usuario interactuar con múltiples escenas o decisiones dentro de una narrativa o juego simple basado en texto y opciones.

El proyecto está diseñado para practicar:
- Estructura semántica en HTML
- Manipulación de DOM con JavaScript
- Gestión de estados y flujo de juego
- Estilos y diseño visual con CSS

---
